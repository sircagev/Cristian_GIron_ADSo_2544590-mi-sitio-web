¿Qué es CSS?

CSS, que significa "Cascading Style Sheets" en inglés, es un lenguaje de hojas de estilo utilizado para describir la presentación y el diseño de documentos HTML (páginas web)

¿Qué aprenderé en este curso?

Diseño y Estilización Web: Con CSS, aprenderás cómo dar estilo a elementos HTML, controlando aspectos como colores, fuentes, tamaños de texto, márgenes, espaciado y más. Esto te permitirá crear diseños visualmente atractivos y coherentes para tus sitios web.

¿Qué es CSS GRID?

es una tecnología de diseño en CSS (Cascading Style Sheets) que permite crear diseños de páginas web de manera más eficiente y flexible.
